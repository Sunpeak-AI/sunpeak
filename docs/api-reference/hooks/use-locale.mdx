## Overview

The `useLocale` hook returns the user's locale string from the host. It is a convenience wrapper around `useHostContext`.

## Import

```tsx
import { useLocale } from 'sunpeak';
```

## Signature

```tsx
function useLocale(app: App | null): string
```

## Parameters

<ResponseField name="app" type="App | null" required>
  The App instance from `useApp()`.
</ResponseField>

## Returns

<ResponseField name="return" type="string">
  BCP 47 locale string (e.g., `'en-US'`). Defaults to `'en-US'` when unavailable.
</ResponseField>

## Usage

```tsx
import { useApp, useLocale } from 'sunpeak';

function MyResource() {
  const { app } = useApp({ appInfo: { name: 'MyResource', version: '1.0.0' } });
  const locale = useLocale(app);

  const formatted = new Intl.NumberFormat(locale).format(1234.56);
  return <p>{formatted}</p>;
}
```

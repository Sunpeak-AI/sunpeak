## Overview

The `useSafeArea` hook returns the safe area insets for handling device notches, status bars, and other system UI. It is a convenience wrapper around `useHostContext`.

## Import

```tsx
import { useSafeArea } from 'sunpeak';
```

## Signature

```tsx
function useSafeArea(app: App | null): SafeAreaInsets
```

## Parameters

<ResponseField name="app" type="App | null" required>
  The App instance from `useApp()`.
</ResponseField>

## Returns

<ResponseField name="return" type="SafeAreaInsets">
  Safe area insets object. Defaults to all zeros when unavailable.

  ```typescript
  {
    top: number;
    bottom: number;
    left: number;
    right: number;
  }
  ```
</ResponseField>

## Usage

```tsx
import { useApp, useSafeArea } from 'sunpeak';

function MyResource() {
  const { app } = useApp({ appInfo: { name: 'MyResource', version: '1.0.0' } });
  const safeArea = useSafeArea(app);

  return (
    <div style={{ paddingTop: safeArea.top, paddingBottom: safeArea.bottom }}>
      Content with safe area padding.
    </div>
  );
}
```
